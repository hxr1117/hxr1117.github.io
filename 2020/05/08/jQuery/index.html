  1. $是jQuery中的顶级元素，是jQuery的代替。

  2. 在页内的DOM元素渲染完再执行。

  3. DOM对象/jQuery对象

    1. 用原生js获取来的对象就是DOM对象，用jQuery获取来的对象就是jQuery对象。

    2. jQuery是利用$对DOM对象包装后产生的对象（伪数组形式储存）。

    3. jQuery对象只能使用jQuery方法，DOM对象只能使用原生的JavaScript属性和方法。

    4. 有些方法jQuery没有，这个时候需要将jQuery对象转换成DOM对象。

      1. DOM对象->jQuery对象

        $("video")

      2. jQuery对象->DOM对象

        $("video")[0].play();

        $("video").get(0).play();

  4. 选择器

    4. 删除类

    5. 切换类
