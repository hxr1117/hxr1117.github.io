Git操作

发表于 2019-08-16 更新于 2020-06-05

Git 安装配置

用户信息

配置个人信息

$ git config --global user.name “username”

$ git config --global user.email youremailaccount

文本编辑器

$ git config --global core.editor

差异分析

$ git config --global merge.tool vimdiff

查看配置信息

$ git config --list

Git基本操作

  1. git status查看当前状态
  2. git add .将所有文件加入缓存区
  3. git commit -m “message”提交到本地仓库
  4. git branch查看分支
  5. git checkout -b new_branch新建分支并切换到新分支
  6. git checkout master切换到主分支
  7. git merge branch_name合并分支到主分支
  8. git push推送到远程仓库
  9. git push -u origin branch_name将分支推送到远程仓库

Git 工作流程

  • 克隆Git资源作为工作目录
  • 在克隆的资源上添加或修改文件
  • 更新资源
  • 提交前的查看修改
  • 提交
  • 撤回修改再次提交
