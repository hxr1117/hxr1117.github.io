Flex布局

发表于 2020-05-02 | 分类于 前端

  1. 任何容器都可以使用flex布局。

  2. 父盒子设为flex后，子元素的flex，clear和vertical-align失效。

  3. 子容器可以横向排列也可以纵向排列。

  4. flex子元素默认一行排列，如果放不下就缩小子元素。

  5. 常见属性

    1. 父项

      1. Flex-direction：设置主轴方向。
        1. 主轴侧轴取决于代码，主轴剩下的就是侧轴。
        2. row/column
      2. Justify-content：设置主轴上子元素的排列方式。
        1. Flex-start/Flex-end/space-around/space-between/center
      3. Flex-wrap
        1. 子元素放不下就换行。
      4. Align-content：设置侧轴上子元素排列方式（多行）。
      5. Align-items：设置侧轴上子元素排列方式（单行）。
      6. Flex-flow:：相当于同时设定了Align-content和Align-items
        1. flex-flow: row wrap;

    2. 子项

      1. flex属性定义子项分配剩余空间。

      3. Align-self：控制子项自己在侧轴上的排列方式。

      4. order：定义项目的排列顺序，数字越小越靠前，默认0。

  4. 通过修改html里面的文字大小改变页面中的元素大小，可以整体控制。

  5. screen和and不能省略

  6. 媒体查询

    2. Mediatype: full//screen

    3. 可以根据不同的屏幕尺寸来改变不同的样式。

移动web

发表于 2020-04-29 | 分类于 前端
  1. meta视口标签
  • <meta name="viewpoint" content="width=device-width, user-scale=no, initial-scale=1.0, maximum-scale=1.0, minimun-scale=1.0">

  1. 物理像素是分辨率，开发时的px和像素在移动端不一定一一对应。

  2. 移动端方案

    1. 单独制作移动端
      1. 流式布局（百分比布局）
      2. flex弹性布局
      3. less+rem+媒体查询布局
      4. 混合布局
    2. 响应式
      1. 媒体查询
      2. bootstrap

  3. 特殊样式

    1. CSS3盒子模型，不管加不加border，padding都不会撑大盒子
    • box-sizing: border-box;
    1. 清除点击高亮
    • -webkit-tap-highlight-color: transparent;
    1. 清除浏览器默认外观，使用自定义样式
    • webkit-appearance: none;
    1. 禁用长按页面弹出菜单
    • img,a { -webkit-touch-callout: none; }

  4. 二倍精灵图

    1. 在firework里把精灵图等比缩小一半，然后再测量位置。
    2. 代码里的background-size也要调成一半。

CSS权重

发表于 2020-04-28 | 分类于 前端

总结：

  1. !important
  2. 行内样式
  3. ID选择器 100
  4. class/属性选择器/伪类选择器 10
    1. 伪类选择器: :active :focus等选项.
  5. 标签选择器/伪元素选择器 1
    1. 伪元素选择器： :before :after

⚠️不同等级没有可比性，比如多少个属性选择器都不能比ID选择器权重高。

建议：

  1. 避免使用!important
  2. 利用id增加选择器权重
  3. 减少选择器个数（避免层层嵌套）
