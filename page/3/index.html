JavaScript基础笔记

发表于 2020-04-26 | 分类于 前端

  1. 严格模式：采用具有限制性JavaScript变体的一种方式，从而使代码显示地 脱离“马虎模式/稀松模式/懒散模式“（sloppy）模式。

    1. “use strict”;

  2. 数据类型：Number/字符串/布尔值/比较运算符/null/undefined/数组/对象/变量

  3. 数组

    1. slice()切片
    2. unshift()往数组头部加元素
    3. shift()删除首元素
    4. splice()从指定索引开始删除并增加元素
    5. concat()拼接，返回新数组

  4. 对象

    1. JavaScript的对象是一种无序的集合数据类型，它由若干键值对组成

    2. 通过.访问属性

    3. 用delete可删除属性

  5. Map/Set

    1. Map = python.dic 键值一一对应
    2. Set = python.tuple 不过可以用delete删除元素

  6. For … of 来迭代可迭代的

  7. 函数

    1. 函数定义：

    3. JavaScript允许传入任意个参数而不影响调用，因此传入的参数比定义的参数多也没有问题，虽然函数内部并不需要这些参数

    4. arguments获得传进函数的所有参数

    5. rest获得额外参数

    6. 方法

      1. 在一个对象中绑定函数，称为这个对象的方法。

      3. 要保证.this指向正确，写法是obj.func()。不然直接调用函数指向的是全局对象window，得到undefined的结果。

      4. 解决方法：

        1. apply()

          1. getAge.apply(xiaoming, []); // 25, this指向xiaoming, 参数为空

          2. apply()把参数打包成Array再传入。

        2. call()

          1. call()把参数按顺序传入。

  8. 高阶函数

    1. Map

    2. Reduce

      1. 效果：[x1, x2, x3, x4].reduce(f) = f(f(f(x1, x2), x3), x4)

    3. filter

    4. sort

      1. 先把array里的元素变成String，再排序，就是按ascii排序。

      2. [10, 20, 1, 2].sort(); // [1, 10, 2, 20]

    5. array

      1. every()方法可以判断数组的所有元素是否满足测试条件。

      2. find()方法用于查找符合条件的第一个元素，如果找到了，返回这个元素，否则，返回undefined。

      3. findIndex()和find()类似，也是查找符合条件的第一个元素，不同之处在于findIndex()会返回这个元素的索引，如果没有找到，返回-1。

      4. forEach()和map()类似，它也把每个元素依次作用于传入的函数，但不会返回新的数组。forEach()常用于遍历数组，因此，传入的函数不需要返回值。
